# Job Tracker

Application de suivi de candidatures (Job Applications Tracker).

## Stack
- Frontend : Vue 3 + Vite
- Backend : Django + DRF (à venir)

## Fonctionnalités (MVP)
- Ajouter une candidature
- Suivre les candidatures par statut (pipeline)
- Modifier / supprimer une candidature

## Lancer le frontend
```bash
cd frontend
npm install
npm run dev
